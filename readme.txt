Запустить встроенный веб-сервер в PHP
php -s localhost:9000 public/index.php

В новом проект нужно проинициализировать composer
composer init

Установить библиотеку для dev
composer require phpunit/phpunit --dev

Решить проблему с иероглифами
В файл .bashrc добавить строки
LANG="ru_RU.UTF-8"; export LANG
LC_ALL="ru_RU.UTF-8"; export LC_ALL
