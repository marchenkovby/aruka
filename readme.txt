Команда для запуска проекта в Lando
lando start

Есть нужно создать файл конфигурации для lando
lando init

lando - это обертка для Docker

Узнать информацию о логинах и паролях в lando
lando info

Перебилдить lando
lando rebuild

Если появляется ошибка при команде lando start,
помог сброс до заводских настроек

Остановить lando
lando stop
